**BrandFactory** is an open-source Brand Operating System for marketers, creators, and small teams.

Every brand gets a **Workspace** — the living home for its identity: target audience personas, brand voice and personality, values, positioning, visual guidelines, messaging frameworks, and whatever else matters. Some brands arrive fully formed; others start as rough ideas. BrandFactory handles both.

Inside each workspace, you run **Projects**: focused spaces where you ideate, iterate, and finalize the work. A project pairs an agent chat with a freeform canvas — rich text, bullets, images, files, moodboard-style snippets. The agent sees your brand context and whatever is on the canvas in real time. You brainstorm side-by-side, pin the ideas worth keeping, filter down to a shortlist, and promote the winners into your finalized brand guidelines.

Projects can be fully freeform, or they can use standardized templates for common jobs — for example, a minimalist social media content calendar. BrandFactory is deliberately un-opinionated about specific workflows. There is no dedicated "naming agent" or "copy agent" — just one brainstorming surface that can produce taglines, content calendars, packaging concepts, or anything else, because the brand context always travels with it.

Where lightweight native tooling makes sense, we build it in. Where depth matters — publishing, production design, analytics — we integrate with best-in-class tools rather than reinventing them.

Self-hosted, privacy-first, modular, and fully extensible, BrandFactory puts you in complete control — no vendor lock-in, no recurring fees, and the freedom to run it with local models or your own stack. One brand context. Endless consistent creative.
